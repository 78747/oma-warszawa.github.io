# OMA Restaurant Website

Statyczna strona restauracji OMA przygotowana do publikacji np. przez GitHub Pages.

## Pliki

- `index.html` - strona główna
- `menu.html` - menu
- `opinie.html` - opinie
- `lokalizacja.html` - lokalizacja
- `kontakt.html` - kontakt
- `styles.css` - style strony
- `script.js` - animacje i interakcje
- `assets/` - zdjęcia i grafiki

## Publikacja na GitHub Pages

1. Wrzuć zawartość tego folderu do głównego katalogu repozytorium.
2. Na GitHubie wejdź w `Settings` -> `Pages`.
3. Ustaw źródło jako `Deploy from a branch`, branch `main`, folder `/root`.
4. Zapisz ustawienia i poczekaj na wygenerowanie linku.
